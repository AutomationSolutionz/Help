How to create a User

  1. From the Menu bar, go to the Admin then Create User option.

  1. Go to the Create/Edit User page.

  2. Fill up the informations like:

    • Full Name,
    • Email,
    • Designation,
    • Username,
    • Password,
    • Confirm password,
    • Teams,
    • Policy.

  1. Multiple of Teams can also be selected.
  2. To remove any required Team, click on the Cross sign (X) of that team.

  1. Click on the Submit button.

  1. Update the informations (If it is needed).
  2. Click on the Update button.
