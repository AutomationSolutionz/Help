How to link Requirement to a Bug

  1. From the Menu bar, go to the Project then Everything.

  1. Go to the All Items page.
  2. Click on the Requirements tab.

  1. Click on a specific Requirement ID.

  1. Go to the Create/Edit Requirement page.
  2. Click on the Bugs tab of the page.

  1. Click on the LINK NEW button of the page.

  1. Select a specific Bug.
  2. Multiple number of Bugs can also be selected.
  3. Click on the LINK SELECTED button.

  1. Write the Description (If it is needed).
  2. Click on the Save button.

  1. In the same process Combined, Tasks and Test cases can also be linked.
